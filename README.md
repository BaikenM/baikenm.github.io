# Developer Portfolio

A clean, responsive static portfolio website built for GitHub Pages.

## Features

- **Semantic HTML5** structure
- **Responsive design** for mobile, tablet, and desktop
- **Tailwind CSS** for modern styling
- **Minimal JavaScript** for smooth interactions
- **Easy to customize** and extend

## Sections

1. **Home** - Welcome message and introduction
2. **Projects** - Showcasing personal and client work
   - Personal Projects: Job Helper, Dombrapp, Threadmap, linqr
   - Client Projects: Shanyraq
3. **About** - Developer bio and expertise
4. **Contact** - Email and GitHub links

## Customization

Before deploying, update the following placeholders:

- Contact email in the Contact section
- GitHub profile URL
- Project links (currently set to `#`)
- Personal information in the About section
- Project descriptions and tech stacks as needed

## Deployment

This site is ready to deploy on GitHub Pages. Simply push to the `main` branch and enable GitHub Pages in your repository settings.

## Technologies

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
